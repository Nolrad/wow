Attumen le Veneur

Résumé

  • 2 tanks (Minuit puis Attumen / boss monté)
  • Dispels malédiction : Mage + Druide
  • Shadow Cleave : boss dos au raid
  • À 25% : fusion + reset menace
  • P3 : stack derrière à < 8 yards pour éviter la charge

Phases

Phase 1 — Minuit (100% → 95%)

  • Tank 1 sur Minuit, raid derrière, laisse la menace monter.

Phase 2 — Arrivée d’Attumen

  • Tank 2 récupère Attumen immédiatement, boss dos au raid (cleave).
  • Dispell Intangible Presence dès que possible (prio tanks).

Phase 3 — Fusion à 25%

  • Reset menace : stop DPS/heal 1–2 sec, le tank reprend.
  • Tout le raid stack derrière le boss à < 8 yards pour éviter Berserker Charge.

Moroes

Moroes est un des boss de Karazhan. Le combat se joue en une seule phase, avec 4 invités choisis parmi 6 possibles : chacun a ses compétences et change la priorité de kill.

Résumé du combat

  • 1 phase : Moroes + 4 adds.
  • Contrôle : prévoir de contrôler 2 adds pendant que vous tuez les 2 autres.
  • Contrôles recommandés : Prêtre (Entrave des morts-vivants) et Chasseur (Piège givrant).
  • Dispel poison requis : Druide / Chaman / Paladin (pour Cécité).
  • Offtank : doit rester 2e à l’aggro pour reprendre quand Moroes lance Suriner.

Plan simple

  1. Au pull : contrôlez 2 adds, tuez 2 adds (burst + CD dès le début).
  2. Ensuite :
    • Option A : tuez Moroes en gardant les contrôles actifs, puis terminez les adds.
    • Option B : finissez tous les adds, puis Moroes (plus long, souvent plus safe si contrôle instable).

Priorité de kill (adds)

  1. Baronne Dorothea Millstipe — Brûlure de mana : à tomber très vite.
  2. Dame Catriona Von’Indi — Soins supérieurs (à kick) + Mot de pouvoir : bouclier (à dispel).
  3. Dame Keira Berrybuck — Bouclier divin + Lumière sacrée (à kick).
  4. Baron Rafe Dreuger — stuns + boost dégâts : souvent à contrôler si besoin.
  5. Seigneur Robin Daris — Frappe mortelle (-50% soins reçus) + Tourbillon : souvent à contrôler.
  6. Seigneur Crispin Ference — désarme + interruptions + gros mur défensif : généralement à contrôler jusqu’à la fin (souvent après Moroes).

Compétences de Moroes

  • Disparition → Garrot : saignement très violent pendant ~5 minutes sur une cible aléatoire.
  • Cécité : désoriente une cible — poison (à dispel).
  • Suriner : désoriente le tank et le boss passe sur la 2e menace → l’offtank doit reprendre.

Gestion de Garrot

  • Prévoir des CD défensifs + soins dédiés sur la cible.
  • Certains immunes/cleanse peuvent retirer Garrot : Bloc de glace, Bouclier divin, Bénédiction de protection, Forme de pierre (nain).

Conseils rapides (exécution)

  • Burst au pull : la partie la plus dure = les 20–30 premières secondes. Utilise les CD pour tomber les 2 premiers adds vite.
  • Contrôle propre : annonce un timer de refresh (Entrave / Piège) et évite les cleaves/AoE sur les adds contrôlés.
  • Placement : pack Moroes + adds “à tuer” pour cleave, mais gardez les adds contrôlés loin du splash.
